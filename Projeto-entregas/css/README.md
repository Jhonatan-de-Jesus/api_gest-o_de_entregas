# 🚚 Flash Logística - Sistema de Entregas

## 📌 Descrição
Sistema simples para gerenciamento de entregas utilizando Supabase.

Permite:
- Criar entregas
- Listar entregas
- Filtrar por status
- Atualizar status
- Deletar entregas

---

## 🧱 Tecnologias
- HTML
- CSS
- JavaScript
- Supabase

---

## ⚙️ Configuração

1. Substitua no código:

SUA_URL_SUPABASE_AQUI
SUA_CHAVE_SUPABASE_AQUI


2. Execute o arquivo `index.html`

/projeto

---

## 🚀 Funcionalidades

### ✔ Criar entrega
Preencha os campos e clique em "Criar Entrega"

### ✔ Listar entregas
Exibe todas ou filtradas por status

### ✔ Atualizar
Botão altera status para "em rota"

### ✔ Deletar
Remove entrega do sistema
